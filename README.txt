INTRODUCTION
------------

Drupal 8 module that provides weather forecast for US cities from a text field formatter by using OpenWeatherMap Api.

-- INSTALLATION --

* Install as you would normally install a contributed Drupal module. See:
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.

CONFIGURATION
-------------

Once the module has been installed, navigate to 
admin/config/services/us_weather_forecast
(Configuration > Web Services > US Weather Forecast through the administration panel)
and configure the api key.

Choose Weather Forecast formatter for your text field
